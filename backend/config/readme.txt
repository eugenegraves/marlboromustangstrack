This directory is for storing your Firebase service account key.

1. Create a Firebase project at https://console.firebase.google.com/
2. Go to Project Settings > Service accounts
3. Click "Generate new private key"
4. Save the downloaded JSON file in this directory as "serviceAccountKey.json"

Important: Never commit this file to version control. It contains sensitive credentials.
The .gitignore file is already set up to exclude this file.
